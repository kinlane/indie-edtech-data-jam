Basic Website Content Demo

This is a working demo to help understand the evolution from web to API, showing website return HTML for humans, and RSS, XML, JSON, and other machine readable content is meant for other application, systems, computers, and mobile devices to use.

  • Blog (HTML) (JSON) (XML) (RSS)
  • Products (HTML) (JSON) (XML) (RSS)
  • Services (HTML) (JSON) (XML) (RSS)

You can view the actual site that runs this at http://website.apievangelist.com. It is not meant to be a complete website or API, just showing how the same URL, can return content, data, and other resources for use in many different scenarios.
