Tools

Some tools that will help you engage with the APIs I've covered, and begin mapping out the APIs you already use, and the life bits you are exchanging on the web daily.

Postman Postman on Chrome is the most efficient way to test, develop and document APIs. Create complex requests, go back in time and view results in a beautiful way. Postman has more than 80 features to help you at every step.
StopLight StopLight provides engineering teams with a new way to document, test, and build web APIs. Our technology easily and automatically maps an API, and then provides tooling and services to reduce development time, and increase reliability and accessibility.
Charles Proxy Charles is an HTTP proxy / HTTP monitor / Reverse Proxy that enables a developer to view all of the HTTP and SSL / HTTPS traffic between their machine and the Internet. This includes requests, responses and the HTTP headers (which contain the cookies and caching information).
