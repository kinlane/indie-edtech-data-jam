Indie EdTech and The Personal API

I wanted to pull together a toolbox that I could use throughout the conversations I would be having at Davidson College this week, that would help folks understand what an APIs, their relationship to the web and mobile world we depend on each day.

Let's look at the world of APIs around you:

  • Why APIs? - You are using APIs every day to pay for things at the store, taking pictures with Instagram, all the way to banking online. The more you understand this world of APIs, the more successful you will be in your personal, and professional career, empowering you to to take more control over more of your digial self.
  • What is a website? - A website is a folder on a computer that is plugged into the Internet, that is given an Internet address, with a domain or subdomain that allows anyone with the address to find it on the Internet. (learn more >>)
  • What is an API? - A API is just like a website, except that instead of returning HTML for humans, it returns RSS, XML, JSON, and other machine readable formats meant for other applications. (learn more >>)
  • The History of APIs - Web APIs have a history, a little shorter than that of the web itself, beggining in 2000, from a a PHD students doctoral thesis. (learn more >>)
  • The Programmable Web - APIs are not just about serving up content, data and other resources for use in applications and devices. HTTP allows APIs to be used for reading, as well as writing, making the web programmable, not just something we consume. (learn more >>)
  • The University API - In 2016, APIs are making their way onto campus in several ways. Sometimes it is campus IT leading the way, others come from vendors selling software on campus, from faculty making it part of their regular work or courses, and is also being brought in by the students. (learn more >>)
  • The Personal API - Amidst all of this, there is a new concept emerging that is centered around you. It is being dubbed the personal API, leveraging an awareness of all things APIs, allowing you to take more control over yourself, the content and data you generate online, in your course work, and in your future career. (learn more >>)
  • Some APIs In Your World - Let's explore just a handful of APIs that are making an impact in your world, and learn about some of their details. (learn more >>)

This workshop runs as a simple HTML, CSS, JavaScript website that runs as a Github repository, using Jekyll and Github Pages. You can fork the project, and play with any of the examples here on this, or your own version running in your Github account.
